# Citation-Formater
You may always have challange with editing your citations using mendeley in academic papers. I write this code to reformat your references as ELSEVIER guide for authors.
The format Ive used is specified for "Corrosion science" journal, which you can see how does it asks you to format your references and make changes based on whatever journals you want to write (keep in mind that most of elsevier's journals follow a similar path in inserting references.

https://www.sciencedirect.com/journal/corrosion-science/publish/guide-for-authors#writing-and-formatting-references
